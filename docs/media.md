# Médias (Supabase)

## Configuration `.env`

- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `SUPABASE_STORAGE_BUCKET`

## Page admin

`/admin/medias`

Fonctions :
- upload (images)
- suppression simple / bulk
- tri (date / nom / taille)
- vue compacte

## Recommandations

- Créer des dossiers logiques : `logos`, `affiches`, `photos`.
